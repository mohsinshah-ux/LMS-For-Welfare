# User Flow Diagrams (Text)

## Loan Officer

Login -> Dashboard -> Create Customer -> Upload KYC Docs -> Create Financing Application -> Submit -> Track Verification -> Generate Agreement -> Monitor Installments

## Manager

Login -> Dashboard -> Review Queue -> Validate Risk/Docs -> Approve/Reject -> Assign Loan/Recovery Officers -> Monitor Team Performance

## Recovery Officer

Login -> Overdue Accounts -> Add Recovery Note -> Schedule Follow-up -> Field Visit/Legal Notice -> Record Recovery -> Close Case

## Super Admin

Login -> Global Dashboard -> Manage Users/Roles -> Configure Products/Workflows -> Monitor Cashflow + P&L -> Review Audit Logs -> System Settings
